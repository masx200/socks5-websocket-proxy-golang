  `SelectConnection` 方法
- 修复代码：
  ```go
  if selector, ok := s.selector.(*upstream.UpstreamSelector); ok {
  ```

- 修复方案：使用 `golang.org/x/net/proxy.SOCKS5`

  ```go
  // 使用golang.org/x/net/proxy.SOCKS5处理认证
  ```
